A **API Key** é a forma de autenticação principal utilizada para acessar rotas protegidas do `HaldenHub`. Cada chave é única, associada a um usuário da plataforma, e pode ser gerada diretamente no seu painel após login.

## 🔑 Como obter sua API Key

1. Acesse o [painel do HaldenHub](https://www.haldenhub.com.br).
2. Faça login com sua conta.
3. No menu lateral, clique em **API Key**.
4. Clique em **Gerar chave**.

Uma chave será gerada no formato: sk-rQiYxSxAdIvJpemBGWnxoemdgyHIWQxs

> 🚨 **Importante:**  
> Guarde sua chave com **segurança**. Ela funciona como uma **senha** e dá acesso total às funcionalidades da sua conta.  
> Não compartilhe essa chave publicamente e **nunca** a exponha em repositórios ou arquivos de frontend.

## 🔒 Como usar a API Key

A API Key deve ser enviada no cabeçalho (`header`) da sua requisição HTTP, no padrão **Authorization: Bearer**.

### Exemplo de requisição

```http
GET /client/user HTTP/1.1
Host: haldenhub.com.br/api/v1
Authorization: Bearer sk-rQiYxSxAdIvJpemBGWnxoemdgyHIWQxs
```


## 🚧 Segurança e limites

- Cada requisição feita com sua chave reduz sua cota de uso, conforme o plano escolhido.
- O uso da chave é monitorado.
- Você pode revogar sua chave a qualquer momento pelo painel e gerar uma nova sempre que precisar.
