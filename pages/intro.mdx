Seja bem-vindo à documentação oficial do **simpleAPI** — uma plataforma moderna de autenticação de usuários, criada e pensada para desenvolvedores. Aqui você encontrará tudo o que precisa para integrar autenticação segura, 2FA e gestão de usuários nas suas aplicações, de forma rápida e clara.

## 🔒 O que é o simpleAPI?

O **simpleAPI** é uma plataforma de autenticação que oferece login seguro, autenticação em dois fatores (2FA), controle de sessões e muito mais. Com APIs REST bem definidas, exemplos práticos e documentação objetiva, é possível integrar recursos avançados de segurança em poucos minutos.

Ideal para aplicações SaaS, produtos mobile, e sistemas corporativos que precisam de uma base sólida de autenticação.

## 💻 API simples e direta

- Baseada em padrões REST
- Respostas em JSON padronizadas
- Exemplos prontos para integração
- SDKs em breve

Nosso foco é te ajudar a **começar rápido**, com o mínimo de fricção e o máximo de clareza.

## 🚀 Como começar

1. Acesse [simpleapi.com.br](https://www.simpleapi.com.br) e crie sua conta gratuita.
2. Gere sua chave de API no painel.
3. Consulte os exemplos de integração nesta documentação.
4. Implemente os fluxos de login, registro, verificação e autenticação em dois fatores.

## ⚙️ Recursos disponíveis

- Cadastro e login de usuários
- Autenticação segura com JWT ou autenticação em dois fatores (2FA via TOTP)
- Invalidação e gerenciamento de sessões

## ✨ Em breve

- Login social (Google, GitHub e outros)
- SDKs para Node.js, Go e React
