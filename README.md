# HaldenHub Docs

HaldenHub is a platform designed to simplify LGPD compliance for websites and SaaS applications, providing tools for cookie consent management, privacy policies, and compliance reporting.

Este diretório contém a documentação técnica (Mintlify), incluindo guias de integração e referência de API.
